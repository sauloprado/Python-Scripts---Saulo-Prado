# Email Profissional

Projeto de automacao para envio de e-mails com apresentacao mais elegante, suporte a HTML e anexos.

## Objetivo

Criar uma base reutilizavel para:

- enviar e-mails com layout profissional
- anexar arquivos automaticamente
- padronizar mensagens recorrentes
- reduzir trabalho manual no envio de documentos

## Primeira Versao

A ideia inicial deste modulo e permitir:

- definicao de remetente, destinatario e assunto
- corpo em HTML com visual limpo
- anexo de um arquivo local
- envio por SMTP

## Evolucoes Possiveis

- leitura de destinatarios por planilha
- templates diferentes por tipo de mensagem
- assinatura personalizada
- historico de envios
- envio em lote
- interface simples no terminal
